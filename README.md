## LangGraph's quistart

## Travel assistant

[travel_assistant.ipynb](travel_assistant.ipynb) mock-implements functionalities such as flight status check, reservation modification, and retrieval augmented generation.
Does so by using tools, short-memory checkpoints, long-memory user preferences, Human-In-The-Loop interruptions, dynamic prompt personalisation, and multi-agent architecture for complex task delegation.
